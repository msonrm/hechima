🧽 へちまラボ luffa lang labo

設定不要で、新配列と本気の日本語変換が動くオンライン日本語入力ラボ — powered by hechima

実験室: ローマ字入力 ・ 薙刀式 ・ フリック入力

hechima（へちま）は、日本語入力の一式 — 配列・かな漢字変換・学習・辞書 — を OS から引き取って、ブラウザのページの中に置くためのソフトウェア群です。 ここ「へちまラボ」はその実験場。各ページが、OS の IME ではできない （あるいは、しにくい）入力のデモになっています。

まずはこの下のエディタで、ふだんどおりローマ字で打って Space で変換してみてください。 ふだんどおりに動くのに、OS の IME を使っていない — それがこのページの実験です。 未確定の下線も、候補窓も、学習も、ぜんぶこのページが自前で描いています。

このページを開いたまま打鍵してください（OS 側の日本語 IME はオフ = 英数直接入力で）。 変換・学習・文書の保存はすべてブラウザ内で完結し、打鍵は一切外部へ送信されません。 文書はこの端末に自動保存され、確定した変換は学習されて候補順に反映されます （「学習リセット」で学習を消去、「クリア」で文書を消去）。 学習・ユーザー辞書・文書は、ラボ内の実験ページ間で共有されます。

  • Space = 変換 / 次候補、↑↓ = 前候補/次候補、数字 1-9 = 直接選択（クリックでも可）
  • 先頭候補でさらに ↑（または Shift+Space）= 追加候補（ひらがな/カタカナ）を展開
  • Shift+英字 = 英単語をそのまま入力（Space で 大文字/小文字/全角 の綴り変換）
  • Enter = 確定、Esc = 戻す、←→ = 文節移動、Shift+←→ = 文節伸縮、Ctrl+BS = 確定取り消し（学習も巻き戻し）
  • エディタ: クリック / 矢印キーでカーソル移動、Ctrl+Z / Ctrl+Y = 文書の undo / redo、貼り付けはプレーンテキスト化
  • 確定済みテキストを選択して 変換キー / Ctrl+/ = 再変換（よみに開き直して選び直す）
  • タッチ端末では「フリック」ボタンで 12 キーフリック入力（詳しくはフリック入力のページへ）
