薙刀式 v18

薙刀式（大岡俊彦さん設計）は、文章を書くための同時打鍵かな配列です。核にあるのは 「相互シフト」— キーを押している間だけ、他のキーの意味が変わる状態ベースの仕組みで、 文節伸縮や確定などの編集操作まで同時打鍵に含まれます。

OS 側でこれを使うには専用の常駐ソフトが必要で（しかも OS ごとに別物）、 iPad の物理キーボードに至っては手段がありません。このページでは配列エンジンごと ブラウザの中にあるので、URL を開くだけ。iPad + 物理キーボードでも動きます — それがこのページの実験です。

OS 側の日本語 IME はオフ（英数直接入力）にして、お使いのキーボード種別（JIS / US）を 選んでから打鍵してください。変換・学習・ユーザー辞書・文書は トップページと共有です。

  • 単打はかな、キーを押しながら別キーを打つと裏面のかな（相互シフト）。押しっぱなしで連続シフト
  • Space = 変換 / 次候補、Enter = 確定、Esc = 戻す、数字 1-9 = 直接選択
  • 薙刀式の編集同時打鍵も効きます: space+T / space+Y（文節伸縮）、M+V（確定）など
  • 配列定義は v18。配列そのものの解説は作者・大岡俊彦さんの ブログへ
