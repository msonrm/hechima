ゲームパッド入力（日本語）

VR/AR でのテキスト入力はまだ未解決の課題です。ゲームパッドならボタンの触覚 フィードバックがあり、フリック配列を応用した同時押しで日本語を組めます。 左手（D-pad ±LB）で子音行、右手（RB/X/Y/B/A）で母音を選び、同時押しで 1 文字。 変換・学習・辞書は OS の IME なしでページ側で動きます。

ゲームパッドを接続してボタンを押すと、下に操作ガイドが出ます。物理キーボードとも 併用できます（下のエディタにフォーカスがあるときだけゲームパッド入力が有効）。

ゲームパッド必須。変換・学習・ユーザー辞書・文書はラボ内の実験ページ間で共有です。

  • かな = 左手で子音行（D-pad、LB で は〜わ行）＋ 右手で母音（RB=あ, X=い, Y=う, B=え, A=お）の同時押し
  • LT = 拗音（か→きゃ）/ 小書き、LT+RT = っ、RT = ん
  • 右スティック: ↑ 濁点/半濁点、→ 長音ー、← 削除、↓ 、→。→空白
  • 左スティック: ↓ 変換/次候補、↑ 前候補、←→ 文節移動、RT+←→ 文節伸縮
  • LS 押込み 確定・改行、RS 押込み 取消、Start 戻す（確定の取り消し。もう一度で文書 undo）
