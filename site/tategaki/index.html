縦書きで書く（検証中）

web に「縦書きで書けるエディタ」はほぼ存在しません。表示だけなら CSS が 縦書きに対応して久しいのに、書く方が成立しなかった最大の理由は OS の IME です — 未確定文字の表示も変換候補の窓も横書きの世界の住人で、縦の文章に割り込んできます。

このページでは IME がまるごとブラウザの中にあるので、未確定文字は本文と一緒に 縦に組まれ、候補窓も縦組で行の左隣に出ます — それがこのページの実験です。 入力はローマ字、物理キーボード向けです。

OS 側の日本語 IME はオフ（英数直接入力）にしてから打鍵してください。変換・学習・ ユーザー辞書・文書はトップページと共有です — 横書きで書いた文書が、 そのまま縦になって開きます。

  • ローマ字で入力、Space = 変換 / 次候補、Enter = 確定、Esc = 戻す
  • 変換中の矢印キーは見た目の向きに一致: ←→ = 候補の段の移動、 ↑↓ = 文節の移動、Shift+↑↓ = 文節の伸縮
  • 候補窓は縦組・右から左（縦書きの読み順）。?cand=lr を付けると 番号 1-9 付き・左から右の並びに切り替わります（検証用の比較）
  • このページは検証中です。表示や挙動は予告なく変わります
